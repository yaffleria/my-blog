# 환영합니다! 👋

안녕하세요! 이 블로그의 첫 번째 한국어 포스트에 오신 것을 환영합니다.

## 이 블로그의 특징

이 블로그는 **혼합형 다국어 블로그**로 운영됩니다. 즉, 어떤 글은 한국어로, 어떤 글은 영어로 작성됩니다.

### SEO 최적화

각 포스트는 다음과 같은 방식으로 SEO가 최적화되어 있습니다:

1. **1개 글 = 1개 언어 원칙**
   - 한 포스트 안에서 언어를 섞지 않습니다
   - 각 글은 처음부터 끝까지 한 언어로 작성됩니다

2. **동적 언어 설정**
   - 한국어 글: `<html lang="ko">`
   - 영어 글: `<html lang="en">`
   - 검색엔진이 정확한 언어로 색인합니다

3. **URL 슬러그 전략**
   - 제목: 한글 블로그에 오신 것을 환영합니다
   - URL: `/blog/welcome-korean-blog` (영어)
   - 링크 공유 시 깨지지 않습니다

## 기술 스택

이 블로그는 다음 기술들로 만들어졌습니다:

- **Next.js 15** - React 프레임워크
- **Tailwind CSS 4** - 유틸리티 우선 CSS
- **Contentlayer** - MDX 콘텐츠 관리
- **Pretendard Variable** - 한글 가변 폰트

## 컬러 테마

블로그의 Primary 컬러는 **오렌지(#FFA500)**로 설정되어 있으며, 다음과 같은 팔레트를 사용합니다:

- `#FFA500` - Primary
- `#DB9D00` - Primary 500
- `#B38000` - Primary 600
- `#614500` - Primary 800

라이트/다크 모드 모두를 지원하며, oklch 색상 공간을 사용하여 일관된 색감을 제공합니다.

## 앞으로의 콘텐츠

이 블로그에서는 다음과 같은 주제들을 다룰 예정입니다:

- 웹 개발 튜토리얼
- 기술 리뷰
- 개발 경험 공유
- 프로젝트 소개

## 마무리

이 블로그가 여러분께 유용한 정보를 제공할 수 있기를 바랍니다.
앞으로 더 많은 고품질 콘텐츠로 찾아뵙겠습니다!
