# Mastercard 2025 Q3 Earnings Components

This directory contains React components for visualizing Mastercard's Q3 2025 earnings data.

## Components

### MastercardRevenueBreakdown

- **Purpose**: Displays revenue composition and growth rates
- **Data**:
  - Basic payment revenue: 60% of total, +12% YoY
  - Value-added services: 40% of total, +25% YoY

### MarketShareChart

- **Purpose**: Shows global card payment market share
- **Data**:
  - Visa: 61%
  - Mastercard: 26%
  - Others: 13%

### GrowthMetricsChart

- **Purpose**: Compares growth rates across key metrics
- **Data**:
  - Total revenue: +17% YoY
  - Value-added services: +25% YoY
  - Cross-border transactions: +15% YoY
  - Basic payment revenue: +12% YoY

## Usage

Import components in your MDX file:

```mdx
<MastercardRevenueBreakdown />
<MarketShareChart />
<GrowthMetricsChart />
```

## Data Source

Based on Mastercard's Q3 2025 earnings report and presentation materials.
